# Automatic Design of Intelligent Systems: Applied to Chess

Bachelor Thesis – Sapienza University of Rome

Author: Matt Clifford Trinidad Ramos

Supervisor: Prof. Enrico Tronci

Academic Year: 2025/2026

---

## Overview

This thesis investigates the application of Quality-Guaranteed (QG) Learning to the domain of chess.

The objective is to design intelligent chess-playing agents that satisfy predefined performance requirements while providing statistical guarantees on their behavior under uncertainty.

The proposed framework combines:

- Evolution Strategies (ES) for controller optimization
- Statistical Model Checking (SMC) for probabilistic verification
- Sequential estimation techniques
- Counterexample-guided refinement
- Neural-network-based decision making

The result is an integrated Quality-Guaranteed learning loop capable of improving controller performance while continuously verifying compliance with specified requirements.

---

## Research Questions

This work explores the following questions:

1. Can a lightweight neural controller trained via Evolution Strategies satisfy scenario-based win-rate requirements?

2. Can Statistical Model Checking provide meaningful confidence-controlled guarantees within realistic computational budgets?

3. Does counterexample-guided refinement improve robustness against unseen scenarios?
